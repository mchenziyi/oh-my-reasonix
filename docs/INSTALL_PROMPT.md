# Prompt for Reasonix

Paste the following request into a Reasonix session:

> Install and configure oh-my-reasonix in the current project. Read and follow
> the pinned installation instructions at
> https://raw.githubusercontent.com/mchenziyi/oh-my-reasonix/main/docs/INSTALL.md
> . Run the dry-run first, show me the plan, and ask before persisting any
> existing user Prompt. Do not modify global configuration, API keys, or the
> Reasonix binary. Finish by running `omr doctor` and report the result.
